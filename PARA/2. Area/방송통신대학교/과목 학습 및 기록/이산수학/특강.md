#### 조건명제 & 쌍조건명제
잘 알아둬야함 조건에 따른 결과값 추정을 다시 계산해보자

- 조건 : 𝑝 → 𝑞
- 쌍조건 : 𝑝 ↔ 𝑞 = (𝑝 → 𝑞) ∧ (𝑞 → 𝑝)

#### 항진명제&모순명제
항진명제 예시 : 𝑝 ∨ ~𝑝

# 함수

용어 : 정의역, 공역-치역, 전사, 단사 등을 알아놓자.

- 전사 : 공역 전체가 선택 받은 함수
- 단사 : 각 결과값의 요소들이 단 하나 씩만 선택 받은 함수
- 전단사 : 전사 ∧ 단사
